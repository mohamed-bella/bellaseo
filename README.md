# SEO Automation SaaS Platform

A high-performance SEO automation platform built with Next.js 16, Turbopack, and Supabase.

## 🚀 Quick Start

### Development
To start the development server from the root:
```bash
npm run dev
```

### Production Build
To build and start the production server:
```bash
npm run build
npm start
```

## 📂 Project Structure

- **`/frontend`**: The core Next.js application (App Router, Tailwind CSS).
- **`/database`**: SQL migrations and database utility scripts.
  - `/migrations`: Sequential SQL migration files.
  - `/scripts`: Database maintenance and migration utilities.
- **`/docs`**: Project documentation and architecture diagrams.
- **`/nginx`**: Nginx configuration for production deployment.

## 🛠 Features

- **Rich Text Editor**: Custom Tiptap-based editor for SEO-optimized content.
- **Campaign Management**: Organize SEO efforts into campaigns and clusters.
- **Keyword Research**: Advanced keyword tracking and metrics.
- **Site Management**: Connect and manage multiple WordPress sites.
- **Enterprise RBAC**: Role-based access control for team collaboration.
